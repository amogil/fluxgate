# Command Line Interface

CLI1. Invoking the executable without arguments must start the proxy using the default configuration location.

**Tags:** `cli`, `config-loading`

CLI2. A `--config` flag must be available to override the configuration path when desired.

**Tags:** `cli`, `config-loading`

CLI3. A `--help` flag must be available to display usage information and available commands.

**Tags:** `cli`

CLI4. Aside from the flags mentioned above, the CLI must not expose any additional commands or subcommands.

**Tags:** `cli`
